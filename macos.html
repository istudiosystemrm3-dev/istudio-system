🛠️ เครื่องมือสำหรับช่าง (โหลด OS & สร้าง USB ในตัว)

แก้ปัญหา Mac รุ่นใหม่ไม่ยอมให้โหลด macOS รุ่นเก่า! แอปเหล่านี้จะดึงไฟล์ตรงจากเซิร์ฟเวอร์ Apple และช่วยสร้าง Bootable USB (แฟลชไดรฟ์ล้างเครื่อง) ให้ในคลิกเดียว

🌟

Mist (Mac OS Installer Super Tool) - แนะนำสำหรับ Mac!

แอป Open Source ยอดฮิตของช่าง Mac ทั่วโลก โหลด OS ได้ทุกเวอร์ชัน เลือกได้ว่าจะโหลดเป็น .app, .iso หรือ สั่งทำ Bootable USB ลงแฟลชไดรฟ์ได้จากในแอปเลย (คอมพิวเตอร์ที่ใช้รันโปรแกรมนี้ต้องเป็น macOS 12 Monterey ขึ้นไป)

⬇️ ดาวน์โหลด Mist (GitHub)
💽

macUSB (เครื่องมืออเนกประสงค์ Mac / Windows / Linux)

โปรแกรมสุดเจ๋งสำหรับสายช่าง! โหลด OS แท้ได้ในตัว และแก้ปัญหาทำ USB รุ่นเก่าบนชิป M1/M2/M3 (ข้ามผ่านใบรับรองหมดอายุ) แถมยังรองรับการทำ Bootable USB ให้ Windows 11 พร้อมตั้งค่า Bypass TPM 2.0 / Secure Boot ได้อัตโนมัติ

⬇️ ดาวน์โหลด macUSB (GitHub) ℹ️ อ่านข้อมูลเพิ่มเติม

📥 แหล่งโหลดไฟล์ติดตั้งโดยตรง (InstallAssistant.pkg)

หากคุณไม่ต้องการใช้แอปช่วยโหลด คุณสามารถโหลดไฟล์ติดตั้งของ Apple (ไฟล์ .pkg) ได้จากเว็บที่ช่างทั่วโลกไว้วางใจ ซึ่งรวบรวม Direct Link แท้จากเซิร์ฟเวอร์ Apple ไว้ครบทุกเวอร์ชัน

💡 วิธีใช้งาน:
1. เข้าเว็บไปโหลดไฟล์ .pkg ของเวอร์ชันที่ต้องการ
2. ดับเบิลคลิกติดตั้งไฟล์ .pkg
3. ระบบจะแตกไฟล์ตัวติดตั้งสีเหลี่ยมๆ (เช่น Install macOS Sonoma.app) ไปวางไว้ในโฟลเดอร์ Applications (แอปพลิเคชัน) ให้ทันที
4. นำตัวติดตั้งนั้นไปสร้าง Bootable USB ต่อได้เลย
🌐 เปิดเว็บไซต์ Mr. Macintosh

💻 วิธีทำ Bootable USB ผ่าน Terminal (เผื่อฉุกเฉิน)

หากคุณโหลดไฟล์ .pkg มาติดตั้งจนได้ไฟล์ .app อยู่ในโฟลเดอร์ Applications แล้ว และต้องการสั่งทำ USB ด้วยตัวเอง ให้ทำตามขั้นตอนนี้ครับ:

1. เตรียมแฟลชไดรฟ์
เสียบแฟลชไดรฟ์ขนาด 16GB ขึ้นไป เปิดแอป Disk Utility เลือกแฟลชไดรฟ์แล้วกด Erase ให้ตั้งชื่อว่า MyVolume (ตัวพิมพ์ใหญ่-เล็กต้องเป๊ะ)
เลือก Format: Mac OS Extended (Journaled) และ Scheme: GUID Partition Map
2. เปิดแอป Terminal แล้วก๊อปปี้คำสั่งไปวาง
macOS Sequoia (15)
sudo /Applications/Install\ macOS\ Sequoia.app/Contents/Resources/createinstallmedia --volume /Volumes/MyVolume
macOS Sonoma (14)
sudo /Applications/Install\ macOS\ Sonoma.app/Contents/Resources/createinstallmedia --volume /Volumes/MyVolume
macOS Ventura (13)
sudo /Applications/Install\ macOS\ Ventura.app/Contents/Resources/createinstallmedia --volume /Volumes/MyVolume
macOS Monterey (12)
sudo /Applications/Install\ macOS\ Monterey.app/Contents/Resources/createinstallmedia --volume /Volumes/MyVolume
