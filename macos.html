🛠️ ขั้นตอนที่ 1: การเตรียม USB Flash Drive

1. เสียบแฟลชไดรฟ์ขนาด 16GB ขึ้นไป เข้ากับเครื่อง Mac (หากเป็นเวอร์ชันใหม่ๆ แนะนำ 32GB)
2. เปิดแอป Disk Utility
3. เลือกแฟลชไดรฟ์ของคุณ แล้วกดปุ่ม Erase (ลบ)
4. ตั้งชื่อแฟลชไดรฟ์ว่า: MyVolume (สำคัญมาก ต้องพิมพ์ตัวพิมพ์ใหญ่-เล็กตามนี้)
5. Format เลือกเป็น: Mac OS Extended (Journaled)
6. Scheme เลือกเป็น: GUID Partition Map (ถ้ามีให้เลือก)
7. กด Erase แล้วรอจนเสร็จสิ้น

💻 ขั้นตอนที่ 2: ดาวน์โหลดและสร้าง Bootable USB (ผ่าน Terminal)

เมื่อโหลดไฟล์ติดตั้งเสร็จแล้ว ให้เปิดแอป Terminal ขึ้นมา ก๊อปปี้คำสั่งด้านล่างไปวาง กด Enter แล้วใส่รหัสผ่านเครื่อง Mac

ระบบปฏิบัติการ (OS) ดาวน์โหลด คำสั่ง Terminal (สำหรับแฟลชไดรฟ์ชื่อ MyVolume)

💡 ทริคสำหรับช่าง (Pro Tip)

หากคุณเปิดหน้าเว็บนี้บนเครื่อง Mac ที่ใช้ Apple Silicon (M1, M2, ฯลฯ) แล้วกดโหลดจาก App Store ไม่ได้ คุณสามารถใช้คำสั่ง Terminal ด้านล่างนี้ เพื่อสั่งให้เครื่องดึงไฟล์ตัวติดตั้งตัวเต็มลงมาที่โฟลเดอร์ Applications ได้ทันทีครับ:

softwareupdate --fetch-full-installer --full-installer-version 14.5

* เปลี่ยนตัวเลข 14.5 เป็นเวอร์ชันที่คุณต้องการดาวน์โหลด
